Сценарии
1. Пользователь наводит мышь на элемент развертки, элемент выделяется.
2. Пользователь кликает на элемент
2.1 Система отправляет асинхронный запрос в Редух
2.1.1 Если возвращается ок, то элемент красится в желтый цвет и добавляется в список элементов.
2.1.2 Если возвращается не ок, то выводится сообщение об ошибке
3. Пользователь кликает на крестик в списке элементов. Система удаляет элемент из списка
4. Пользователь кликает на + или - на развертке. Разверка масштабируется.
